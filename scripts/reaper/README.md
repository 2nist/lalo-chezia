# REAPER Scripts (Album-Specific)

Use this folder for Lua/ReaScript helpers for `lalo-chezia`.

Included starter scripts:

1. `create_track_template.lua`: create standard track/folder structure.
2. `apply_color_map.lua`: apply color conventions by track prefix.
3. `session_qc.lua`: verify required tracks and bus names.
4. `export_stems.lua`: standard stem export helper.

Current status:

1. Scripts are scaffolds (`TODO` placeholders) to be expanded incrementally.
2. Keep behavior aligned with the docs before adding project-specific exceptions.

Keep script behavior aligned with:

- `docs/REAPER_STANDARD.md`
- `docs/REGENERATION_POLICY.md`
