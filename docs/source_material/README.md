# Source Material

This folder contains legacy planning material imported from the `lalo-chezia-20260220T201251Z-1-001.zip` archive.

## Layout

- `songs/<nn_slug>/` contains per-song notes and lyric drafts.
- `album/` contains album-wide docs.
- `image_ideas/` contains artwork concept docs.

## Naming

- Filenames were normalized to lowercase snake_case for cross-platform compatibility.
- Original content was kept; only file/folder names were cleaned.
