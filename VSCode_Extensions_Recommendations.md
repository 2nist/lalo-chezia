# VSCode Extensions for Efficient Styling

This document provides a comprehensive list of VSCode extensions specifically curated for styling React/TypeScript applications with Tailwind CSS and CSS-in-JS.

## 🎨 Core Styling Extensions

### 1. Tailwind CSS IntelliSense
- **Extension ID**: `bradlc.vscode-tailwindcss`
- **Publisher**: Brad Cornes
- **Rating**: ⭐⭐⭐⭐⭐ (4.8/5)
- **Downloads**: 10M+
- **Price**: Free

**Key Features:**
- Real-time class name suggestions
- Hover previews of applied styles
- Linting for unused Tailwind classes
- Support for custom configurations
- IntelliSense for arbitrary values

**Why Essential:**
This is the most critical extension for Tailwind CSS development. It provides intelligent autocomplete that understands your custom configuration and suggests relevant classes as you type.

**Installation Command:**
```bash
code --install-extension bradlc.vscode-tailwindcss
```

## ⚙️ Configuration Tips

### Workspace-Specific Extensions
Create `.vscode/extensions.json` in your project root to recommend extensions to team members:

```json
{
  "recommendations": [
    "bradlc.vscode-tailwindcss",
    "heybourn.headwind",
    "naumovs.color-highlight",
    "esbenp.prettier-vscode",
    "dbaeumer.vscode-eslint"
  ]
}
```

### Extension Settings
Add these settings to your `.vscode/settings.json` for optimal styling experience:

```json
{
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.organizeImports": true,
    "source.fixAll.eslint": true
  },
  "prettier.singleQuote": true,
  "prettier.trailingComma": "es5",
  "editor.rulers": [80, 120],
  "workbench.colorTheme": "One Dark Pro"
}
```
